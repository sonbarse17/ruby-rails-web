# Ruby Rails Web

Ruby on Rails web application with modern MVC architecture.

## Features
- Ruby on Rails framework
- MVC architecture
- SQLite database
- Bootstrap styling

## Quick Start
```bash
docker build -t ruby-rails-web .
docker run -p 3000:3000 ruby-rails-web
```

Visit: http://localhost:3000
